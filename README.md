Pacote oficial do ecossistema Exkyn para envio de e-mails. Este pacote implementa os contratos definidos no núcleo estrutural **[Exkyn.Notifications](https://nuget.org/packages/Exkyn.Notifications)**, garantindo um padrão arquitetural unificado.

## 📦 Instalação

Instale o pacote via .NET CLI:

```bash
dotnet add package Exkyn.Notifications.Email
